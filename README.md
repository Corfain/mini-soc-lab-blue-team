# Mini SOC Lab (Blue Team Home Lab)

This project demonstrates a small-scale Security Operations Center (SOC) lab built using Windows and Linux virtual machines. Logs are collected and analyzed in a SIEM (Wazuh) to simulate real-world detection and monitoring.

---

## 🎯 Objectives
- Centralize logs from multiple systems
- Detect suspicious activities
- Build detection rules
- Practice Blue Team workflows

---

## 🧱 Lab Environment
- Windows Virtual Machine
- Linux Virtual Machine
- Wazuh SIEM
- Sysmon
- System and authentication logs

---

## 🔍 Detection Use Cases
- Brute Force Detection
- PowerShell Abuse Detection
- Privilege Escalation Detection
